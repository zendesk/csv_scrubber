Awesome srubber for csv processing, macro string escaping and stuff

Install
=======

```Bash
gem install csv_scrubber
```

Usage
=====

```Ruby
require 'csv_scrubber'

...

Zendesk::CsvScrubber.scrub_output(value)
```
